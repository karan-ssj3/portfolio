## Autonomous Stock Trading Bot
**Technologies:** PPO, Stable-Baselines3, Reinforcement Learning, Technical Indicators

Built a custom Reinforcement Learning agent using PPO for trading in a simulated market with continuous action spaces and real-world constraints (transaction costs, slippage). Incorporated technical indicators (RSI, MACD, EMA) into the observation space. Engineered a reward function to maximise risk-adjusted returns. Trained over 5M+ timesteps, achieving stable policy convergence with a Sharpe ratio of 0.54. Tuned PPO parameters for exploration-exploitation tradeoffs; evaluated using Sortino ratio, drawdown, and win rate metrics.

## Image Captioning with Deep Learning
**Technologies:** VGG16, LSTM, Flickr8k, PyTorch

Built an image captioning pipeline combining VGG16 for visual feature extraction and LSTM for sequence generation. Applied transfer learning by modifying the CNN architecture. Preprocessed captions with tokenisation, embeddings, and special tokens for sequence alignment. Trained on the Flickr8k dataset, fine-tuning LSTM depth and dropout for optimal convergence. Achieved semantically relevant captions on validation images.
